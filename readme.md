Siwo
======
基于swoole4.2.x版本封装的简易框架

##框架实现了
1. http,tcp,udp,websocket的封装，只需要定义好路由编写对应的控制器即可使用
2. 简易封装了mysql协程客户端

##框架安装
1. git clone url
2. composer install

##框架启动

1. 启动
php siwod start
2. 停止
php siwod stop
3. 重启
php siwod restart
